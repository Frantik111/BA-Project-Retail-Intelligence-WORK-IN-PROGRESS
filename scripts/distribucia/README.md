# Distribucia aplikácie

## V01 - Pre prvotné zistenie informácií z blueetooth
Pre distribúciu sme zvolili zip súbor v ktorom sa nachádza .bat skript, priečinok s datbázou a python skriptom a priečinok v ktorom je winpithon rozbalený

Instalacia.bat doplní do winpithonu pip flask a potom vytvorí do windows start menu odkaz na python skript.
